# UI Assets

Place ici les images utilisees par l'interface (branding / habillage).

Structure recommandee:

- `logo/` : logos
- `titles/` : titres visuels
- `icons/` : icones UI
- `backgrounds/` : images de fond UI

Exemple d'URL dans l'application:

- `/ui-assets/logo/mon-logo.png`
- `/ui-assets/titles/titre-hero.png`
- `/ui-assets/icons/menu-home.png`
- `/ui-assets/backgrounds/header-bg.jpg`
